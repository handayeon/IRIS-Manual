Manual
=========================

IRIS의 기능별 사용 방법을 설명합니다.


==============================  =====================================================================================================================================================================
구분                             설명
------------------------------  ---------------------------------------------------------------------------------------------------------------------------------------------------------------------
IRIS STUDIO                      IRIS Studio를 활용하여 데이터 탐색과 시각화 결과를 원하는 형태로 구성하여 손쉽고 빠르게 미려한 보고서를 생성할 수 있습니다.
IRIS WEB                         빅데이터 솔루션 IRIS의 다양한 기능을 웹을 통해 접근하고 활용할 수 있습니다.
IRIS Discovery Service           여러 데이터소스에 있는 데이터를 사용자가 동일한 명령어로 검색 및 분석 할 수 있도록 해 주는 분석 미들웨어 서비스입니다.
IRIS Database                    IRIS 관리자가 시스템을 효율적으로 운영할 수 있도록 다양한 명령어를 제공하며, 본 문서를 통해 이에 대한 자세한 설명을 확인할 수 있습니다.
==============================  =====================================================================================================================================================================
